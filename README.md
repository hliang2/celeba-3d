CelebA-3D

A Large-Scale 3D In-the-Wild Face Dataset

Note: This repository currently hosts the preview project page. The full paper, codebase, and dataset release are coming soon.

About

CelebA-3D focuses on reconstructing high-fidelity 3D avatars from in-the-wild images with unprecedented detail and topological consistency.

Local Project Page

To view the project webpage locally:

Clone this repository.

Add your input images to the images/ directory (format: 00000.jpg, 00001.jpg, etc.).

Add your rendered 3D avatar videos to the videos/ directory (format: 00000.mp4, 00001.mp4, etc.).

Open index.html in any modern web browser.

(Note: The page includes visual fallbacks, so you can safely open index.html to see the layout even before adding your media files.)
